Brief overview of what HTML is:
-------------------------------
HTML (Hypertext Markup Language) is a
markup language used to create content for
the World Wide Web.
It consists of a set of tags and attributes
that are used to describe the structure and content of a web page. HTML files are typically saved with
a .html or .htm file extension.
